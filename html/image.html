Hub 镜像加速

为了加速 Docker 镜像拉取，使用以下命令设置registry mirror


mkdir -p /etc/

tee /etc//daemon.json <<EOF
{
    "registry-mirrors": ["https://docker.tianjiaji.cloudns.org"]
}
EOF

systemctl daemon-reload

systemctl restart

不用设置环境也可以直接使用，用法示例：


 pull docker.tianjiaji.cloudns.org/library/mysql:8.0

说明：library是一个特殊的命名空间，它代表的是官方镜像。

可以一次性导入多个 DockeHub 镜像源

    tee /etc/docker/daemon.json <<EOF
    {
        "registry-mirrors": [
            "https://hub.uuuadc.top",
            "https://docker.anyhub.us.kg",
            "https://dockerhub.jobcher.com",
            "https://dockerhub.icu",
            "https://docker.ckyl.me",

    EOF
